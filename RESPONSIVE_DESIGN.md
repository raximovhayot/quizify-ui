## Tailwind Classes Breakdown

### Container Responsiveness

```tsx
className = 'container mx-auto px-4 sm:px-6 lg:px-8 py-6 sm:py-8 max-w-7xl';
```

- `container` - Tailwind container utility
- `mx-auto` - Centered horizontally
- `px-4` - 16px horizontal padding on mobile
- `sm:px-6` - 24px on small screens (â‰¥640px)
- `lg:px-8` - 32px on large screens (â‰¥1024px)
- `py-6` - 24px vertical padding on mobile
- `sm:py-8` - 32px on small screens
- `max-w-7xl` - Maximum width of 1280px

### Grid Responsiveness

```tsx
className = 'grid grid-cols-1 lg:grid-cols-3 gap-6 sm:gap-8';
```

- `grid` - CSS Grid layout
- `grid-cols-1` - Single column by default (mobile)
- `lg:grid-cols-3` - 3 columns on large screens (â‰¥1024px)
- `gap-6` - 24px gap on mobile
- `sm:gap-8` - 32px gap on small screens and up

- Better use of horizontal space
